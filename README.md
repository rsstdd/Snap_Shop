# Angular.js SnapShop

Snapshop is a fictitious E-commerce website for a camera retail store. I built this project with Node Express, postgreSQL, Angular, and Angular-Materialize.

##Technologies used include:

* Angular.js
* Node.js
* Express
* postgreSQL
* Knex.js
* Materialize CSS
* Boom
* body-parser
* cookie-parser
* request
* humps
* jsonwebtoken
* Joi
